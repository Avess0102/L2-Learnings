The car is very good.   -   El coche / el carro es muy bueno.

I want   -   (yo) quiero

I want it.   -   Lo quiero.

I don’t want it.   -   No lo quiero.

I want a Coca-Cola.   -   Quiero una Coca-Cola.

I don’t want a coffee. I want a beer.   -   No quiero un café. Quiero una cerveza.

I need   -   necesito

I need it.   -   Lo necesito.

I don’t need it.   -   No lo necesito.

I need a car.   -   Necesito un coche.

I need a beer.   -   Necesito una cerveza.

now   -   ahora

I want it now.   -   Lo quiero ahora.

I want it, but I don’t need it now.   -   Lo quiero, pero no lo necesito ahora.

like that / that way   -   así

It’s like that.   -   Es así.

It’s not like that.   -   No es así.

I need it like that.   -   Lo necesito así.

I don't want it like that.   -   No lo quiero así.

I’m not like that.   -   No soy así.

I’m sorry but it’s not possible like that.   -   Lo siento pero no es posible así.

goodbye   -   adiós

Goodbye and thank you very much.   -   Adiós y muchas gracias.
