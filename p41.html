possible   -   posible

impossible   -   imposible

it is (it’s)   -   es

It’s possible.   -   Es posible.

It’s imposible.   -   Es imposible.

for me   -   para mí

for you   -   para ti

It’s for me.   -   Es para mí.

It’s for you.   -   Es para ti.

It’s possible for me.   -   Es posible para mí.

It’s impossible for you.   -   Es imposible para ti.

a Coca-Cola for me   -   una Coca-Cola para mí.

A coffee for me and a Coca-Cola for you.   -   Un café para mí y una Coca-Cola para ti.

it’s not   -   no es

It’s not for you.   -   No es para ti.

It’s not for me. It’s for you.   -   No es para mí. Es para ti.

the   -   el / la

the Coca-Cola   -   la Coca-Cola

the coffee   -   el café

The Coca-Cola is for me.   -   La Coca-Cola es para mí.

The coffee is for you.   -   El café es para ti.

The Coca-Cola’s for me, and the coffee is for you.   -   La Coca-Cola es para mí, y el café es para ti.

The coffee’s not for me. It’s for you.   -   El café no es para mí. Es para ti.

It's not possible for me   -   No es posible para mí.
